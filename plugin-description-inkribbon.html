🖋️ InkRibbon — IntelliJ Plugin

"A lone typewriter sits quietly in the corner.
Its ink ribbon is intact."
— Resident Evil 2

Amid the chaos of code, between unknown bugs and builds that fail without reason, there is a safe place.
InkRibbon is your typewriter. Your only method of salvation.

This plugin for IntelliJ IDEA is inspired by the legendary save system from Resident Evil 2.
Instead of saving your survival, it lets you save your developer progress.

  • 🗂️ Choose a folder to store your Markdown entries
  • 📅 Automatically create YYYY-MM-DD.md
  • 📝 Built-in editor inside the ToolWindow
  • 💾 One-click save
  • 🧘‍♂️ Clean, quiet interface
